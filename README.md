# RFID-ERPNext-System
Integration of RFID scanning onto ERPNext stocking system


1. The RFID will be used as labels, similar to barcode, and put onto items. 

2. A handheld RFID scanner is used to scan RFID tags/stickers.

3. Then you can do either check in, check out or stock check the scanned items with ERPNext stock.





The handheld RFID scanner used is SR-RU6L50 which is an UHF RFID reader.

The System is built using node-red, ERPNext and Android Studio with MQTT as the messsagging protocol.
